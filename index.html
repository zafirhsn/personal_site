Shooter.io

Shooter.io is a real-time, online multiplayer game that allows users to control a circle and obtain points by defeating other users. Each player controls a circle using the left, right, up, down arrow keys or by using the W, A, S, D keys. A player can use thier mouse to point at another player on the screen and left click in order to fire a laser at the opposing player. If the other player is hit by the laser, our player receives 1 point. Every player has 5 lives and a life is lost when they are hit by a laser. Once a player dies, they are removed from the server and given the option to restart and play again. The game continues for the players who are still alive. Shooter.io was made using Node.js, Express, p5.js, and socket.io. The main goal of the project was to create a real-time multiplayer game that any number of players can join. The client side was created using p5.js in order to render the game on the user's screen. Socket.io was used to send messages between client and server in order to enable real time communication and updates to the game. Node.js and Express were used to create the server with EJS being used as the templating engine.

Statify

Statify is a site that allows Spotify users to see their listening habits and statistics in a clean, user-friendly manner. Listeners can view their top artists and songs of all time, of the last 6 months, and of the last 4 weeks. They can also see how popular their favorite songs and artists are on Spotify on a scale from 1 to 100. Users can also see other song attributes such as the 'danceability' and 'happiness' of their top tracks. Statify does not store any information on users. My main goal in creating this site was to allow Spotify users on any device or browser to be able to log in and view their listening habits. As a result, cross-browser compatibility and a fully responsive design were the top priorities. The site was created with a mobile-first attitude using Bootstrap 4. JQuery's ajax allowed for communnication with Spotify's web API according to the OAuth2 authentication model. This site was made using HTML, CSS, and JavaScript.
